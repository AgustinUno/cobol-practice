## 📋 Overview

This repository contains hands-on COBOL exercises exploring core language features — from basic grading logic to file I/O and stored procedure simulation. Each file is a standalone program targeting a specific concept covered in Programming 3.

> Written as lab and practice exercises — not intended for production use.

---

## 📁 Programs

| File | Description |
|------|-------------|
| `Simple_Grade.COB` | Grade computation logic using COBOL conditionals |
| `File_Handling.cbl` | Reading from and writing to files using COBOL file I/O |
| `File_Manipulation.cbl` | Advanced file operations — update and delete records |
| `Ordering.cbl` | Sorting and ordering data records |
| `Stored_Procedure.cbl` | Simulating stored procedure behavior in COBOL |
| `data.txt` | Sample data file used by the file handling programs |

## 🚀 Getting Started

### Prerequisites

Install GnuCOBOL:
```bash
# Ubuntu/Debian
sudo apt install gnucobol

# macOS (Homebrew)
brew install gnu-cobol
```

### Compile & Run

```bash
# Clone the repo
git clone https://github.com/AgustinUno/COBOL-Practice-.git
cd COBOL-Practice-

# Compile a program
cobc -x Simple_Grade.COB -o simple-grade

# Run it
./simple-grade
```
